# QSL Guide
How to use QSL.
QSL is short for QuickShortcutLanguage, It's a language made for making shortcuts.

Here is a Example.

<pre><code>
#get essentials

start script

table {
    strings {
        createTable("data")
    }
}
data = 
    title
    stored
    app
proceed 1
inside(data)
    title = ("title")
    stored = ("directory")
    app = ("app")
end script
</code></pre>
